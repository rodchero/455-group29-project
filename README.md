## Problem Description
We model a city as a grid with non-uniform population density. Each cell is assigned a value between 0 and 10, where higher values represent denser populations.


The goal is to distribute a fixed set of services, such as schools, emergency facilities, and grocery stores, in locations that minimize population-weighted travel distance with greater priority given to densely populated areas.
